## Run Locally

Clone the project

```bash
  git clone https://github.com/Anoosh-MR/Booking-app--UI.git
```

Go to the project directory

```bash
  cd Booking-app--UI/client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
